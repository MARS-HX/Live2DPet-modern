# Live2DPet v1.5.0 — 双模式 + 独立聊天窗口 + 酒馆预设系统

> **发布版便携 EXE 下载**：见下方附件 `Live2DPet.exe`（解压即用，无需安装）

---

## ✨ 新功能

### 🐱 双模式 AI 聊天
陪伴模式 + 酒馆模式，各自独立记忆，互不干扰

| 模式 | 说明 |
|------|------|
| 🐱 **陪伴模式** | 原有的 AI 宠物聊天，长期记忆，主动感知窗口场景 |
| 🏰 **酒馆模式** | 你是冒险者，AI 是酒馆老板，自定义奇幻世界观 |

### 🪟 独立聊天窗口
- 脱离桌宠窗口限制，可拖到屏幕任意位置
- 单窗口限制，防止多开卡顿
- 支持语音输入、TTS 播报、模式切换

### 📜 酒馆预设系统
- **内置 3 套预设**：奇幻酒馆 / 赛博酒吧 / 仙侠客栈
- **用户自定义**：支持 JSON 格式预设的编辑、导入、导出
- **自动冒险日记**：切换模式时 AI 根据对话生成日记弹窗

### 🔊 多后端 TTS 语音合成
- **小米蜜模 Mimo**（默认，开箱即用）
- **阿里云 TTS**（需开通阿里云语音合成服务）
- **本地 VITS2**（完全离线，需自行部署 VITS2 服务）
- 新增 **TTS 诊断工具**，一键查看服务状态

### 🧠 增强记忆系统
- 对话自动总结（每 10 条对话）
- 兴趣话题追踪与提取
- 用户偏好学习（喜欢/不喜欢/习惯）
- 主动回忆机制
- 酒馆模式独立记忆

### 🎨 全新毛玻璃 UI
- 设置页毛玻璃卡片效果（backdrop-filter: blur）
- **暗色模式**支持，跟随系统偏好
- **完整三语言**（中文 / English / 日本語），覆盖全部设置页

### 🎤 语音交互优化
- 本地 Web Speech API 优先（离线可用）
- PCM 直接录音，不再依赖 WebM 转换
- 按住说话，松开发送
- 实时音量电平指示

---

## 🔧 改进

- 移除废弃的 Translation API 和 VOICEVOX 遗留代码
- 截图频率可配置（设置页自定义间隔）
- 首次运行自动弹出使用说明
- 配置文件版本检测（v1→v2 自动迁移）
- 记忆存储改为合并写入（修复对话与视觉记忆互相覆盖的 bug）
- 所有 JS 文件经过语法验证（52 个文件全部通过）

---

## 🐛 Bug 修复

- 修复 TTS `optimize_text_preview` 参数导致 API 400 错误
- 修复 WebM→WAV 转换时 `Maximum call stack size exceeded` 爆栈
- 修复 `get-active-window` 频繁 fallback 导致的刷屏（添加 2 秒缓存）
- 修复多处字符串换行符被错误写入文件的语法问题
- 修复多后端 TTS 配置保存与加载问题
- 修复 i18n 中 tavern 键缺失导致的界面显示英文
- 修复暗色模式下拉菜单白色背景
- 修复独立聊天窗口模式切换无响应
