# 🐱 Live2DPet 桌面宠物 - 使用说明

> 一款带 AI 聊天的桌面 Live2D 宠物伴侣

---

## 📦 快速开始

### 系统要求
- **Windows 10/11** 64位
- 无需安装任何运行时（已打包为独立 EXE）

### 首次运行
1. 解压 `Live2DPet-Portable.zip`
2. 双击 **`Live2DPet.exe`**
3. 在右下角托盘区找到宠物图标，右键打开**设置**

---

## ⚙️ 基础设置

### 1️⃣ 配置 AI 模型（必须！）
在设置 → **API 设置** 中填写：

| 字段 | 说明 | 示例 |
|------|------|------|
| API 地址 | 你的 AI 接口地址 | `https://openrouter.ai/api/v1` |
| API Key | 你的 API 密钥 | `sk-xxx...` |
| 模型名称 | 使用的模型 | `gpt-4o-mini` / `qwen3.5:2b` |

> **本地部署用户**：如果跑的是 Ollama，地址填 `http://localhost:11434/v1`

### 2️⃣ 配置语音（可选）
在设置 → **TTS 设置** 中配置语音合成，支持多种后端：

| 服务商 | 说明 | 配置项 |
|--------|------|--------|
| **小米蜜模 Mimo** | 云端 API，需申请 Key | API 地址、API Key、风格描述 |
| **阿里云 TTS** | 阿里云语音合成，需开通服务 | AccessKey ID/Secret、AppKey、音色 |
| **本地 VITS2** | 本地运行的 VITS2 服务，无需联网 | 服务地址、接口路径、说话人 ID |

> **VITS2 用户**：确保本地 VITS2 服务已启动并可通过 HTTP 访问
> 默认地址 `http://localhost:7860`，可在设置中自定义

### 3️⃣ 加载宠物模型
在设置 → **模型设置** 中：

1. 模型类型选 **Live2D**
2. 点击 **导入模型文件夹**，选择你的 Live2D 模型目录（包含 `.model3.json` 文件）
3. 可以勾选 **复制到用户目录** 避免路径问题
4. 点击 **保存模型配置**

> 也可以选择**图片模式**，用静态图片作为宠物

## 🎮 基本操作

### 宠物窗口
| 操作 | 效果 |
|------|------|
| **拖拽** | 移动宠物位置 |
| **点击宠物** | 互动（触发 AI 对话） |
| **窗口右下角「−」「+」** | 缩小/放大宠物 |
| **窗口右下角「💬」** | 打开聊天框 |
| **窗口右下角「⚙」** | 打开设置 |
| **窗口右下角「✕」** | 关闭宠物 |
| **右键宠物** | 打开快捷菜单 |

### 💬 文字聊天
1. 点击宠物窗口的 **「💬」** 按钮
2. 在底部输入框打字，按 **Enter** 发送
3. 宠物会用语音 + 气泡回复你

> 快捷回复按钮会自动出现，点击就能快速聊天

### 🎤 语音聊天
1. 点击聊天框的 **「🎤」** 按钮
2. **按住说话**，松开发送
3. 语音会自动转为文字发给 AI

> 语音识别优先使用本地的 **Web Speech API**（无需网络）
> 如果失败会自动切换到云端 Mimo API

### 😊 表情系统
宠物会根据聊天内容自动切换表情和动作：
- 🤗 开心 → 笑脸、光环
- 😢 安慰 → 泪目、抱枕
- 😡 生气 → 生气表情
- 😴 无聊 → 睡觉动画

你可以在设置 → **表情设置** 中自定义每个表情的持续时间和启用状态

## 🧠 记忆系统

宠物会记住你和它的聊天内容：

| 记忆类型 | 说明 |
|---------|------|
| 💬 **对话历史** | 记住最近 300 条聊天记录 |
| 📝 **自动总结** | 每 10 条对话自动生成阶段总结 |
| 🏷️ **话题追踪** | 自动提取你感兴趣的话题 |
| ❤️ **偏好学习** | 从"我喜欢/不喜欢"中学习你的喜好 |
| 👁️ **视觉记忆** | 记住你正在看什么窗口（如"你在逛淘宝"） |

> 所有记忆存储在 `enhance-data.json` 文件中
> 在设置中可随时 **清空记忆**

## 🔧 高级设置

### 检测间隔
控制宠物主动找你的频率（设置 → 基本设置）

| 设置项 | 默认 | 说明 |
|--------|------|------|
| AI 检测间隔 | 60秒 | 多久检测一次你在干什么并主动对话 |
| 回复间隔 | 10秒 | 两条回复之间的最短间隔 |
| 截图间隔 | 0秒 | 0=每次检测都截图，调大减少后台活动 |

### 增强功能
在设置 → 增强设置 中可开启：

| 功能 | 说明 |
|------|------|
| 记忆系统 | 记录窗口使用习惯（已默认开启） |
| 视觉记忆 | 自动截屏并识别画面内容 |
| 知识获取 | 自动搜索并学习相关知识（需联网） |

### 角色卡片
在设置 → **角色设定** 中可自定义宠物的：
- **名字**、**对你的称呼**
- **性格描述**、**说话风格**
- **交互反馈**（点击/拖拽时说什么）
- 支持多角色切换

## ❓ 常见问题

### Q: 宠物窗口怎么不见了？
A: 看看右下角系统托盘，宠物图标在那里。双击或右键可重新打开。

### Q: 语音识别不好用？
A: 确保麦克风权限已开启。Web Speech API 在部分 Windows 版本上可能需要联网。

### Q: TTS 语音合成失败？
A: 检查 **TTS 设置** 中的 API Key 是否正确。可以在设置中点 **「Test TTS」** 测试。

### Q: 怎么换模型？
A: 设置 → 模型设置 → 重新导入文件夹。支持 Live2D Cubism 3/4 模型。

### Q: 聊天记录存在哪里？
A: 在程序目录下的 `enhance-data.json` 文件中。删除它即可清空所有记忆。

### Q: 宠物不主动说话了？
A: 检查**检测间隔**是否太大，或 **AI 模型** 是否配置正确。点 **「Test API」** 测试连接。

| 文件 | 说明 |
|------|------|
| `Live2DPet.exe` | 主程序 |

| `models/` | 本地 Whisper 模型（可选，用于离线语音识别） |
